## 🌟 О проекте

**Screenshot Sender PRO** — это асинхронное Windows-приложение, созданное для геймеров, стримеров и разработчиков. Оно позволяет по нажатию комбинации клавиш моментально сделать снимок экрана и отправить его вашему Telegram-боту, не прерывая игровой процесс или работу.

Главная фишка приложения — **Режим невидимки (Stealth Mode)**. Благодаря использованию Windows API, окно приложения можно сделать абсолютно невидимым для программ записи экрана (OBS Studio, Discord, Skype, захват экрана Windows).

---

## 🔥 Ключевые возможности

*   🚀 **Мгновенная отправка:** Скриншоты сохраняются в оперативную память (без записи на диск) и асинхронно отправляются в Telegram.
*   👻 **Stealth Mode (Невидимка):** Скрывает окно настроек от любых программ захвата экрана. Вы видите окно, а зрители на стриме — нет!
*   ⌨️ **Умные горячие клавиши:** Интерактивная запись макросов. Поддерживает любые комбинации и работает **независимо от языка раскладки** (благодаря считыванию VK-кодов).
*   🛡️ **Защита от мисскликов:** Встроенный анти-спам таймер (Debounce) не позволит отправить 10 скринов, если у вас залипла кнопка.
*   🎨 **Современный Dark UI:** Красивый интерфейс на базе `customtkinter` с плавными анимациями.
*   📝 **Встроенные логи:** Консоль внутри приложения для отслеживания ответов от серверов Telegram в реальном времени.
*   🔄 **Умный буфер обмена:** Встроенный фикс бага Tkinter с кириллицей — `Ctrl+V` и `Ctrl+C` работают идеально на любой раскладке.
*   💻 **Автозагрузка:** Возможность добавить приложение в реестр для скрытого старта вместе с Windows.
